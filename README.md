# Datatub is Coming Soon

## Local development

Install dependencies:

```sh
bun install
```

Run tests:

```sh
bun run test
```

Run linter:

```sh
bun run lint
```

Automatically fix linting errors:

```sh
bun run lint:fix
```

Run tests and linter:

```sh
bun run check
```

Run server:

```sh
bun run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Docker preview

```sh
docker compose up --build
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
